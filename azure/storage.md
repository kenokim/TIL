## Azure storage account
- 최상위 namespace
- 성능: 표준 = hdd, premium = ssd

## Azure storage server

## Azure data lake storage
- 대규모 데이터 분석 및 레이크
- hdfs 호환
- 데이터 웨어하우징
- 기본적으로 비정형 데이터 (blob) 을 저장하기 위함
